# 模拟走盘指南

模拟走盘是把历史比赛按日期顺序重放，从 10,000 元纸面本金开始，观察模型候选信号、组合观察和模拟结算表现。

## 它做什么

1. 每个历史比赛日只使用该日前的数据生成候选信号。
2. 按策略选择单关观察和 2 串 1 组合观察。
3. 使用纸面金额进行模拟结算。
4. 输出最终模拟本金、总盈亏、ROI、命中率、最大回撤和走盘明细。
5. 给出问题诊断和改进建议。

## 它不做什么

本工具只做本地概率分析、纸面模拟和回测诊断。不提供投注、下单、支付、代购或任何自动化购彩能力。

## 如何运行

```bash
python3 -m src.cli.simulate_operation --historical-data data/fixtures/operation_walkforward_sample.csv --initial-bankroll 10000 --format json
```

## 如何解读

- 最终模拟本金：历史复盘结束后的纸面本金。
- 总盈亏：最终模拟本金减初始模拟本金。
- 命中率：已结算观察项命中的比例。
- 组合表现：单关、2 串 1、3 串 1 分开统计。
- 最大回撤：纸面本金从高点到低点的最大跌幅。

模拟盈利不代表未来表现。用户真实 CSV 的质量决定复盘参考价值。
