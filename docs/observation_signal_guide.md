# Observation Signal Guide

候选信号是模型与市场之间出现分歧时的研究观察项，不是购买建议，也不是操作指令。

## 怎么看候选信号

1. 看比赛、玩法方向和赔率。
2. 看市场去水概率。
3. 看模型概率。
4. 看概率差 Edge。
5. 看理论期望 EV。
6. 看风险等级和本地解释。
7. 如果是组合观察，重点看组合命中概率和风险放大说明。

## 什么是观察清单

观察清单用于临时保存你想进一步研究的候选信号，只保存在浏览器内存，不写文件，刷新后可以丢失。

## 重要边界

观察清单不代表购买建议。本工具只做本地概率分析与回测诊断，不提供投注、下单、支付、代购或自动化购彩能力。

概率模型不保证结果。回测结果不保证未来表现。串关会显著放大风险。
