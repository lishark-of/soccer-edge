被拒组合原因

风险解释

为什么当前没有 2串1？因为串关需要多场同时命中。即使每场都有正 EV，组合命中概率也会快速下降。本工具会用相关性折扣和风险上限过滤组合。提高风险档位可能增加纸面收益预期，但也会增加回撤和连续亏损概率。该功能仅用于纸面模拟研究。
